# Finance Analytics Streamlit App

AI-powered finance analytics dashboard built with Streamlit,
Snowflake, Python, dbt, and SQL — featuring AI agents,
semantic data models, prompt engineering, and automated
reporting for Strategic Finance teams.

## Tech Stack
- **Streamlit** — production finance dashboards and apps
- **Snowflake** — data warehouse and semantic layer
- **Python** — AI agents, data pipelines, reporting automation
- **dbt** — data models, tests, documentation, lineage
- **SQL** — advanced queries, window functions, CTEs
- **OpenAI/LLM APIs** — AI-assisted analytics and agents
- **openpyxl** — automated Excel reporting exports
- **Git** — version control, PRs, branching

## Features

### 1. AI Finance Agent
- Prompt-engineered AI agent for revenue analysis
- YAML + Markdown skill files for reusable workflows
- Encodes repeatable finance processes into invokable tools
- Non-technical analysts produce analyst-quality output

### 2. Revenue Analytics Dashboard
- Weekly and quarterly revenue summary pipelines
- ARR, NRR, bookings tracking and visualization
- Sensitivity analysis for quarterly business reviews
- Real-time KPI monitoring and alerting

### 3. Semantic Data Model
- Natural language query interface for finance tables
- Plain English queries against complex financial data
- Cortex Analyst-style semantic layer design
- Reusable component patterns for consistent UI

### 4. Automated Reporting
- Multi-tab Excel exports formatted to spec
- Quarterly earnings cycle automation
- Investor relations data exports
- Automated disclosure and IR data needs
